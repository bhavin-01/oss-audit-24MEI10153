# Project Description

This repository contains five Bash shell scripts created as part of the Open Source Software Capstone Project. The scripts demonstrate basic Linux system administration tasks and automation using Bash scripting.

The scripts were written and tested on a Linux environment and illustrate concepts such as variables, loops, conditionals, command substitution, file handling, and user interaction.

---

# Repository Structure

script1_system_identity.sh – Displays system information such as kernel version, user, uptime and Linux distribution.

script2_package_inspector.sh – Checks whether a specific open-source package is installed and displays information about it.

script3_disk_auditor.sh – Audits important system directories and reports disk usage and permissions.

script4_log_analyzer.sh – Reads a log file and counts occurrences of a specific keyword.

script5_manifesto_generator.sh – Interactively generates a personal open-source manifesto and saves it to a text file.
